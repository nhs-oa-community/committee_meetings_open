# meetings-public
This repository is for sharing minutes of NHS-OA Community Committee meetings.

The minutes are checked to confirm they can be made public and, if necessary, confidential or sensitive information is removed.

The Committee's Terms of Reference can be found in the [NHS-R Way Quarto book](https://nhsrway.nhsrcommunity.com/)

## Any errors?
If you spot an error or have any queries, these can be raised as an issue on the repository or contact the NHS-OA Community Core Team via the contact information in the [NHS-R Way Quarto book](https://nhsrway.nhsrcommunity.com/).
